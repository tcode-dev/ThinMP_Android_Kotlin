# ThinMP_Android_Kotlin

`ThinMP` is simple music player for Android.

## Features

* device music play
* background play
* favorite artists
* favorite songs
* playlists
* shortcuts

## Environments

* Android Studio
* Kotlin
* Jetpack Compose
* Gradle
* minSdkVersion 30
* targetSdkVersion 31
* Google Pixel 4a
* Android Version 12

## Libraries

* Coil - https://github.com/coil-kt/coil
* Realm - https://realm.io/
