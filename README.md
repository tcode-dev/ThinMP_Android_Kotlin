# ThinMP_Android_Kotlin

`ThinMP` is simple music player for Android.

## Features

* device music play
* background play
* favorite artists
* favorite songs
* playlists
* shortcuts

## Environments

* Android Studio
* Kotlin
* Gradle
* minSdkVersion 30
* targetSdkVersion 31
* Google Pixel 4a

## Libraries

* Epoxy - https://github.com/airbnb/epoxy
* Glide - https://github.com/bumptech/glide
* Realm - https://realm.io/
