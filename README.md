## Features

* device music play
* background play
* favorite artists
* favorite songs
* playlists
* shortcuts

## Environments

* Android Studio Dolphin (2021.3.1)
* Kotlin
* Gradle
* minSdkVersion 33
* targetSdkVersion 33
* Android Version 13
* Google Pixel 4a
* Google Pixel 7 Pro

## Libraries

* Accompanist - https://google.github.io/accompanist
